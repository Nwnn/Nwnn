# 望月 龍一 (Ryuichi Mochizuki)
機械学習と分散協調システム  

## 主な研究参加
### テーマ1: 分散機械学習とモデル結合
分散されたノード間で学習した複数の機械学習モデルを、パラメータや重みを統合することで効率的に結合する手法を研究しました。
データを中央集約せずに、従来手法と同等性能を維持できる分散学習環境の実現を目指しました。

#### 貢献内容
* 分散機械学習のシミュレーション環境を構築し、ノード間通信やモデル統合システムまでを自作
* モデル結合アルゴリズムおよび結合対象選定アルゴリズムの考案・議論と実装に関与
* 提案手法アルゴリズムの実装

#### 使用した技術、知識
Python3:    
* GensimのWord2vec内部実装を解析・改変し、分散環境での学習に最適化    
* Numpyによるモデル内部のベクトル演算・重み統合処理    
* Matplotlibによる実験結果の可視化
 
Node.js:  
* TCP通信を用いたノード間プロトコルの試作  
* WebベースGUIを持つクライアントサーバ型アプリケーション開発
  
その他インフラ:   
* 実験用ラックサーバ構築、OS・アプリ導入から運用保守 
* Cisco/Yamahaルータによる研究室ネットワーク・VPN設計・構築
